-----------
--AutoVol
-----------

Autovol es un script que automatiza la fase incial de análisis con Volatility.

Requisitos:
-Tener instalada como mínimo la versión 2.6.1 de Volatility

Instrucciones de uso:
1)Introducir la ruta absoluta del volcado de RAM que se quiera analizar.

2)Introducir los plugins que se quieran emplear en esta fase inicial -separados por espacios-.

3)Seleccionar uno de los perfiles sugeridos introduciendo el número correspondiente a la posición 
que ocupa el perfil, en caso no saberlo con seguridad se recomienda escoger el primero.

4)Se pueden comprobar los outputs de los plugins seleccionados en la carpeta con ruta ~/outputs, 
cada uno von el nombre de su respectivo plugin, así como los perfiles recomendados en el .txt con 
el nombre del perfil seleccionado.

AutoVol is an script that automate the initial phase of analysis with Volatility.

Requirements: 
-Have at least Volatility version 2.6.1 installed 

Instructions for use: 
1)Enter the absolute path of the RAM dump that you want to analyze. 

2)Introduce the plugins that you want to use in this initial phase -separated by spaces-. 

3)Select one of the suggested profiles by entering the number corresponding to the position 
what occupies the profile, if you do not know for sure it is recommended to choose the first one. 

4)You can check the outputs of the selected plugins in the folder with path ~/outputs, each one 
has the name of its respective plugin, as well as the profiles recommended in the .txt with the 
name of the selected profile.
